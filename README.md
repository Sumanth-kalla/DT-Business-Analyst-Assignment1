About This Submission

This repository contains my submission for the DT CultureTech Business Analyst (Clinic Operations & Care Systems) Role Test Drive Assignment.

The assignment focuses on designing practical, low-friction systems that clinics can realistically run to reduce operational errors and protect doctor time.
No software has been built as this submission is intentionally focused on system design, daily workflows, and decision rules, in line with the assignment instructions.
